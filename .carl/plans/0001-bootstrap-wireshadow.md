# 0001 - Bootstrap WireShadow

## Objective
Establish a minimal, safe, and testable foundation for WireShadow with cARL governance and MV3-first architecture.

## Deliverables
- Governance artefacts in `.github/carl/` and `.carl/`
- TypeScript project skeleton
- MV3 extension stubs
- Event model with trust-boundary semantics
- Classifier and redaction baseline
- Google Colab recogniser baseline
- README and targeted tests

## Out of scope
- CDP runtime implementation
- Full notebook parsing
- Production telemetry pipeline
- Any exfiltration or offensive behavior
